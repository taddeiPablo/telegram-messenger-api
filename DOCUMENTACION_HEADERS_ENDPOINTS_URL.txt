PASOS PARA PODER PROBAR EL SERVICIO :
-----------------------------------
url del serivicio a utilizar :[ https://telegram-messenger-api-nyuo.onrender.com ]

para ver la documentacion de la api ir a : /api-docs/


------------------------------------------------------------------------------------------
Registrar Usuarios :
------------------------------------------------------------------------------------------
POST : /users/register

headers : 
	Accept : application/json
	Content-Type : application/json
body : 
{
  "email": "",
  "password": "",
  "name": ""
}

------------------------------------------------------------------------------------------
Login de Usuario :
------------------------------------------------------------------------------------------
POST : /users/login

headers : 
	Accept : application/json
	Content-Type : application/json
body : 
{
  "email": "",
  "password": ""
}

Response : si no hubo errores le tiene que devolver los datos del usuario logueado y un JWT.

------------------------------------------------------------------------------------------
configuracion del mensaje automatico :
------------------------------------------------------------------------------------------
POST : /messages/config
headers : 
	Accept : application/json
	Content-Type : application/json
	Authorization : Bearer <JWT> 

body : 
{
 "autoReply": "Â¡Hola! Gracias por escribirnos."
}

------------------------------------------------------------------------------------------
ver el historial de conversaciones completo : 
------------------------------------------------------------------------------------------
GET: /messages/history

headers : 
	Accept : application/json
	Content-Type : application/json
	Authorization : Bearer <JWT> 

Response : devuelve todas las conversaciones que se llevaron a cabo.

-------------------------------------------------------------------------------------------
enviar mensajes a cualquier usuario :
-------------------------------------------------------------------------------------------
POST : /messages/send
headers : 
	Accept : application/json
	Content-Type : application/json
	Authorization : Bearer <JWT> 

body: 
{
  "chatId": "<receiverId>",
  "text": "Hola, Lei tus mensajes en que puedo ayudarte ?."
}

Response : nos devuelve un status y un mensaje 
{
  "status": "success",
  "message": "Mensaje enviado correctamente a Telegram"
}
